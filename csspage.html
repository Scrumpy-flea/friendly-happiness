What is CSS?

Css stands for Cascading Stylesheets. It's used for adding Styles to your webpage.

Once you are familiar with HTML and you have made your first page, the next step is to create a stylesheet. This is done in the sameway that a HTML page is created. Just make sure you when you save as it's called .css This tells the computer that the text is in the language of CSS.
